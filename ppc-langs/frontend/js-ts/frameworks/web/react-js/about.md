# About

[React](https://react.dev/) – библиотека для разработки Web и нативных интерфейсов. Библиотека позволяет собирать интерфейсы (экраны, страницы, виджеты и целиком приложения) из отдельных кусочков — компонентов.

Компоненты React — это javascript-функции. У компонентов свой язык разметки — JSX или TSX

### React Compiler

React добавил новый dev-инструмент — [React Compiler](https://react.dev/learn/react-compiler/introduction). Он автоматически на этапе сборки оптимизирует компоненты React (memoize components). Таким образом, можно писать компоненты без memoization, а компилятор сам оптимизирует компоненты для перерисовки (re-render).

## React Hooks

Хуки – это объекты, созданные для хранения данных и функций на стороне клиента (user's device memory)
