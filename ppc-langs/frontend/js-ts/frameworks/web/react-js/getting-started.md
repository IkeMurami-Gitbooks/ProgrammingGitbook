# Getting Started

Сайт фреймворка: [https://react.dev/](https://react.dev/)

## Создать проект

### Чистый React

JS:

```
$ npx create-react-app test-project-name
$ cd test-project-name
$ yarn start (or npm start)
```

TS:

```
$ npx create-react-app test-project-name
$ cd test-project-name
$ yarn start (or npm start)
```

### Next.js

Проект создается с поддержкой typescript, по умолчанию.

```
$ npx create-next-app
```

## Разработка
