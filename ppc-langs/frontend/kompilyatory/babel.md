# Babel

Он позволяет писать современный код JavaScript, который будет работать даже в старых браузерах.

Site: [https://babeljs.io/](https://babeljs.io/)

## Транспиляция (transpilation)

Транспилер (transpiler) — программа, которая анализирует код и переводит ее с одного языка на другой (или на тот же), но по функциональность код получается идентичным. По сути, это тип компилятора — source-to-source компилятор

Как это делает babel можно [попробовать тут](https://babeljs.io/repl) и, например, увидеть в какой JS-код превращается наш React-компонент.
