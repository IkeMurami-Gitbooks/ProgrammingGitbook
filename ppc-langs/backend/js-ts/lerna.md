# Lerna

Это инструмент, который позволяет управлять пакетами в монорепозитории:

```
my-lerna-repo/
  package.json
  packages/
    package-1/
      package.json
    package-2/
      package.json
```

В общем, суперпопулярная штука у разработчиков

Ссылка: [https://github.com/lerna/lerna](https://github.com/lerna/lerna)
