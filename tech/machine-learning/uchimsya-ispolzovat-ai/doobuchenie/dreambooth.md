# DreamBooth

DreamBooth — это метод персонализации text-to-image диффузионных моделей (например, Stable Diffusion), позволяющий «научить» модель конкретному объекту/персонажу по небольшому набору его фото (обычно 5–20), чтобы затем генерировать новые изображения этого же объекта в любых сценах и стилях.
