# ControlNet

ControlNet — это архитектурное расширение для диффузионных моделей (например, Stable Diffusion), которое позволяет «привязывать» генерацию к дополнительным структурным подсказкам: контуру, позе, глубине, сегментации и т. п. Итог — модель точнее соблюдает композицию, позы, макет, при этом сохраняет творческую вариативность текста.
