# API Proxies

Предоставляют единый интерфейс к моделям

* OpenRouter: [https://openrouter.ai/models?order=pricing-low-to-high](https://openrouter.ai/models?order=pricing-low-to-high)
  * Есть бесплатные модели (gpt oss, mistral, xAI, deepseek), то есть бесплатны со стороны платформы, а сами модели oss
  * Ограничения: 50 запросов в день
* HuggingFace
* SegMind: [https://www.segmind.com/](https://www.segmind.com/)
  * Специализируется на генерации медиа контента (изображений, видео)
  * За 10$ можно попробовать, есть месячные тарифы
  * По сути подняли модели у себя и продают доступ
* Together: [https://www.together.ai/](https://www.together.ai/)
