# FLUX

Black Forest Labs [https://bfl.ai](https://bfl.ai)

HuggingFace: [https://huggingface.co/black-forest-labs](https://huggingface.co/black-forest-labs)

FLUX — это семейство генеративных моделей для создания изображений по текстовому описанию (text-to-image, а также image-to-image ). Это современные диффузионные/DiT‑модели, обученные на лицензированных и синтетических данных, с упором на качество, точное следование промптам и хорошую типографику

Есть открытые веса (dev-версия)
